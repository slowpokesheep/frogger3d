## Controls

* `W` - Translate y+ 
* `A` - Translate x- 
* `S` - Translate y- 
* `D` - Translate x+ 
* `Q` - Translate z+
* `E` - Translate z-
* `MouseWheel` - Scroll up and down to zoom in and out (z+, z-)
* `Mouse` - Click down and move around to rotate

### Debug

* `P` - Pause the game
* `O` - Single step while game is paused
